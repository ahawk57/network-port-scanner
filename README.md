# Network Port Scanner 🔍

A Python-based network security tool for scanning open and closed ports on any host.

## Features
- Scan common ports or enter custom ports manually
- Real-time color coded terminal output
- Automatic report saved to scan_results.txt
- Port labels identifying what each service does

## How to run
```python
python scanner.py
```

## Technologies
- Python 3
- Socket programming
- ANSI terminal colors

## Example output
[OPEN]   Port 80  — HTTP - Web Traffic
[closed] Port 21  — FTP - File Transfer

Only scan hosts you have permission to scan. scanme.nmap.org is provided for legal practice.
